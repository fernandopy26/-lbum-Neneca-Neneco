# Album Neneca & Neneco

Site-presente com contador do tempo juntos, album mensal em formato de livro e cartao 10x15 cm para imprimir com QR code.

## Como adicionar fotos

- Fotos do album: coloque em `imagens/meses/AAAA-MM.jpg`, por exemplo `imagens/meses/2026-06.jpg`.
- Foto do cartao impresso: `imagens/cartao-foto.jpg`.
- QR code do cartao impresso: `imagens/qr-code.png`.

O site cria uma pagina nova automaticamente quando entra um novo mes.
